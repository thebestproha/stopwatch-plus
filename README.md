# Stopwatch Plus ⏱️

A modern, persistent timer application with multiple countdown timers that auto-start and save your progress.

## Features

- ✅ Multiple concurrent timers
- ✅ Auto-start on creation (60-minute default)
- ✅ Background persistence - timers keep running even when you close the browser
- ✅ Local storage - all data saved automatically
- ✅ Clean, modern UI with Tailwind CSS

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React Icons

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
